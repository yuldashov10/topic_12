–§–∞–∫—Ç–∏—á–µ—Å–∫–∏, —ç—Ç–æ –ø—Ä–æ—Å—Ç–æ –æ–±—Ä–∞—Ç–Ω–∞—è —Å—Ç–æ—Ä–æ–Ω–∞ –ø–æ–Ω—è—Ç–∏—è _–ø–æ–¥–º–Ω–æ–∂–µ—Å—Ç–≤–∞_.

–Ω–µ —Ö—Ä–∞–Ω–∏—Ç –ø–æ–∑–∏—Ü–∏–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –≤ –æ—Ç–ª–∏—á–∏–µ, –Ω–∞–ø—Ä–∏–º–µ—Ä, –æ—Ç —Å–ø–∏—Å–∫–æ–≤.

```python
colours: set[str] = {
    "—á–µ—Ä–Ω—ã–π", "–±–µ–ª—ã–π", "–∂—ë–ª—Ç—ã–π", "–∫—Ä–∞—Å–Ω—ã–π", "—Å–∏–Ω–∏–π", "–∑–µ–ª—ë–Ω—ã–π"
```

**–¢–æ–ª—å–∫–æ —Ö—ç—à–∏—Ä—É–µ–º—ã–µ (–Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–µ) —ç–ª–µ–º–µ–Ω—Ç—ã**

–ú–Ω–æ–∂–µ—Å—Ç–≤–∞ –º–æ–≥—É—Ç —Ö—Ä–∞–Ω–∏—Ç—å —Ç–æ–ª—å–∫–æ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–µ (—Ö—ç—à–∏—Ä—É–µ–º—ã–µ) —Ç–∏–ø—ã: —á–∏—Å–ª–∞, —Å—Ç—Ä–æ–∫–∏, –∫–æ—Ä—Ç–µ–∂–∏, **–Ω–æ –Ω–µ —Å–ø–∏—Å–∫–∏** –∏ –¥—Ä. –∏–∑–º–µ–Ω—è–µ–º—ã–µ
—Ç–∏–ø—ã. –ü–æ–ø—ã—Ç–∫–∞ –¥–æ–±–∞–≤–∏—Ç—å –∏–∑–º–µ–Ω—è–µ–º—ã–π —Ç–∏–ø –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –ø—Ä–∏–≤–µ–¥—ë—Ç –∫ –æ—à–∏–±–∫–µ `TypeError: unhashable type`.

```python
digits_ru: set[tuple[int, str]] = {
    (1, "–æ–¥–∏–Ω"),
    (2, "–¥–≤–∞"),
    (3, "—Ç—Ä–∏"),

    [4, "four"],
    [5, "five"],
    [6, "six"],

```

**–°–æ–∑–¥–∞–Ω–∏–µ –ø—É—Å—Ç–æ–≥–æ –º–Ω–æ–∂–µ—Å—Ç–≤–∞**

–ï—Å–ª–∏ –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –ø—É—Å—Ç—ã—Ö —Å–ø–∏—Å–∫–æ–≤ –∏ –∫–æ—Ä—Ç–µ–∂–µ–π –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–∫ —Å–∫–æ–±–∫–∏ `[]`, `()`, —Ç–∞–∫ –∏ —Ñ—É–Ω–∫—Ü–∏–∏ `list()`, `tuple()`,
—Ç–æ —Å –º–Ω–æ–∂–µ—Å—Ç–≤–∞–º–∏ –≤—Å—ë —á—É—Ç—å –∏–Ω–∞—á–µ. –°–æ–∑–¥–∞—Ç—å –ø—É—Å—Ç–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –º–æ–∂–Ω–æ —Ç–æ–ª—å–∫–æ —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `set()`.

> –í–∞–∂–Ω–æ: –ó–∞–ø–∏—Å—å —á–µ—Ä–µ–∑ `{}` —Å–æ–∑–¥–∞—ë—Ç –Ω–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, –∞ –ø—É—Å—Ç–æ–π —Å–ª–æ–≤–∞—Ä—å (–∏–∑—É—á–∏–º –≤ —Å–ª–µ–¥—É—é—â–µ–π —Ç–µ–º–µ).


–ú–Ω–æ–∂–µ—Å—Ç–≤–∞ –≤ Python —Ä–∞–±–æ—Ç–∞—é—Ç –±—ã—Å—Ç—Ä–æ –∏ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ. –≠—Ç–æ –æ—Ç–ª–∏—á–Ω—ã–π –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç, –∫–æ–≥–¥–∞ –Ω—É–∂–Ω–æ –∏—Å–∫–ª—é—á–∏—Ç—å –¥—É–±–ª–∏–∫–∞—Ç—ã –∏–ª–∏ –≤—ã–ø–æ–ª–Ω–∏—Ç—å
–ª–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤—Ä–æ–¥–µ –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è, –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏—è –∏–ª–∏ —Ä–∞–∑–Ω–æ—Å—Ç–∏.

## –ú–µ—Ç–æ–¥—ã –º–Ω–æ–∂–µ—Å—Ç–≤

–£ –º–Ω–æ–∂–µ—Å—Ç–≤–∞ —Ç–∞–∫–∂–µ –µ—Å—Ç—å —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–µ –º–µ—Ç–æ–¥—ã, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ–∑–≤–æ–ª—è—é—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å —Ç–∏–ø–æ–≤—ã–µ –¥–µ–π—Å—Ç–≤–∏—è: –¥–æ–±–∞–≤–ª–µ–Ω–∏–µ, —É–¥–∞–ª–µ–Ω–∏–µ,
–æ–±–Ω–æ–≤–ª–µ–Ω–∏–µ –∏ –¥—Ä. –æ–ø–µ—Ä–∞—Ü–∏–∏. –ù–∏–∂–µ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –∫–∞–∂–¥—ã–π –∏–∑ –º–µ—Ç–æ–¥–æ–≤ –±–æ–ª–µ–µ –ø–æ–¥—Ä–æ–±–Ω–æ.

`set.add()` - –¥–æ–±–∞–≤–ª—è–µ—Ç –æ–¥–∏–Ω —ç–ª–µ–º–µ–Ω—Ç –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–æ.

–ú–µ—Ç–æ–¥ `add()` –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –æ–¥–∏–Ω –∞—Ä–≥—É–º–µ–Ω—Ç –∏ –¥–æ–±–∞–≤–ª—è–µ—Ç –µ–≥–æ –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–æ. –ï—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç —É–∂–µ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç, –æ–Ω –ø—Ä–æ—Å—Ç–æ –∏–≥–Ω–æ—Ä–∏—Ä—É–µ—Ç—Å—è.

`set.update()` - –¥–æ–±–∞–≤–ª—è–µ—Ç —Å—Ä–∞–∑—É –Ω–µ—Å–∫–æ–ª—å–∫–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∏–∑ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞.

–†–∞–±–æ—Ç–∞–µ—Ç –∫–∞–∫ `add()`, –Ω–æ –¥–æ–±–∞–≤–ª—è–µ—Ç **–≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã** –∏–∑ –ø–µ—Ä–µ–¥–∞–Ω–Ω–æ–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ (—Å–ø–∏—Å–∫–∞, –∫–æ—Ä—Ç–µ–∂–∞, —Å—Ç—Ä–æ–∫–∏ –∏ —Ç.–¥.).

```python
new_data: list[str] = ["–±–∞–Ω–∞–Ω", "–≥—Ä—É—à–∞"]

fruits: set[str] = {"—è–±–ª–æ–∫–æ", "–∞–π–≤–∞", "–∞–Ω–∞–Ω–∞—Å"}
```

`set.remove()` - —É–¥–∞–ª—è–µ—Ç —É–∫–∞–∑–∞–Ω–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç.

–ï—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç–∞ –≤ –º–Ω–æ–∂–µ—Å—Ç–≤–µ –Ω–µ—Ç, —Ç–æ –≤–æ–∑–Ω–∏–∫–Ω–µ—Ç –æ—à–∏–±–∫–∞ `KeyError`.

`set.pop()` - —É–¥–∞–ª—è–µ—Ç –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç.

–í—ã —É–∂–µ —Å—Ç–∞–ª–∫–∏–≤–∞–ª–∏—Å—å —Å –º–µ—Ç–æ–¥–æ–º `list.pop()` —É —Å–ø–∏—Å–∫–æ–≤, –º–µ—Ç–æ–¥ —É–¥–∞–ª—è–ª –ø–æ—Å–ª–µ–¥–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–ª –µ–≥–æ. –¢–∞–∫ –∫–∞–∫
–º–Ω–æ–∂–µ—Å—Ç–≤–∞ –Ω–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—ã–µ, –Ω–µ–ª—å–∑—è –∑–∞—Ä–∞–Ω–µ–µ –∑–Ω–∞—Ç—å –∫–∞–∫–æ–π —ç–ª–µ–º–µ–Ω—Ç –±—É–¥–µ—Ç —É–¥–∞–ª—ë–Ω. –ï—Å–ª–∏ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –ø—É—Å—Ç–æ–µ –≤–æ–∑–Ω–∏–∫–Ω–µ—Ç `KeyError`.

```python
emoji: set[str] = {"üé≤", "üéØ", "üß©", "üé≥"}
```

`set.copy()` - —Å–æ–∑–¥–∞—ë—Ç –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–Ω—É—é –∫–æ–ø–∏—é –º–Ω–æ–∂–µ—Å—Ç–≤–∞.

–¢–∞–∫–∂–µ —Ä–∞–±–æ—Ç–∞–µ—Ç –∫–∞–∫ —É —Å–ø–∏—Å–∫–æ–≤, –ø–æ–∑–≤–æ–ª—è–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å —Å –∫–æ–ø–∏–µ–π –æ–±—ä–µ–∫—Ç–∞, –Ω–µ –∏–∑–º–µ–Ω—è—è –æ—Ä–∏–≥–∏–Ω–∞–ª.


---

–î–∞–ª–µ–µ —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º –º–µ—Ç–æ–¥—ã, –æ—Å–Ω–æ–≤–∞–Ω–Ω—ã–µ –Ω–∞ –∫–ª–∞—Å—Å–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏—è—Ö –∏–∑ —Ç–µ–æ—Ä–∏–∏ –º–Ω–æ–∂–µ—Å—Ç–≤.

`set_a.union(set_b)` - –æ–±—ä–µ–¥–∏–Ω—è–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–∞.

–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–∑ –æ–±–æ–∏—Ö –º–Ω–æ–∂–µ—Å—Ç–≤. –í–º–µ—Å—Ç–æ –º–µ—Ç–æ–¥–∞ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –æ–ø–µ—Ä–∞—Ç–æ—Ä `|`,
—Ä–µ–∑—É–ª—å—Ç–∞—Ç –±—É–¥–µ—Ç –∏–¥–µ–Ω—Ç–∏—á–Ω—ã–º. –û–¥–Ω–∞–∫–æ –ø—Ä–µ–¥–ø–æ—á—Ç–∏—Ç–µ–ª—å–Ω–µ–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–µ—Ç–æ–¥—ã, –æ–Ω–∏ —è–≤–Ω–æ –æ—Ç—Ä–∞–∂–∞—é—Ç —Å–º—ã—Å–ª –≤—ã–ø–æ–ª–Ω—è–µ–º–æ–π –æ–ø–µ—Ä–∞—Ü–∏–∏
–∏ –ø–æ–≤—ã—à–∞—é—Ç —á–∏—Ç–∞–µ–º–æ—Å—Ç—å –∫–æ–¥–∞.

```python

b: set[int] = {3, 4, 5, 6}

res_2: set[int] = a | b

print(res_2)  # {1, 2, 3, 4, 5, 6}
```

`set_a.intersection(set_b)` - –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤.

–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ, —Å–æ–¥–µ—Ä–∂–∞—â–µ–µ —Ç–æ–ª—å–∫–æ –æ–±—â–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã. –ê–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω–æ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –æ–ø–µ—Ä–∞—Ç–æ—Ä `&`

`set_a.difference(set_b)` - –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ä–∞–∑–Ω–æ—Å—Ç—å –º–Ω–æ–∂–µ—Å—Ç–≤.

–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ –∏–∑ —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É—é—â–∏—Ö –≤ `set_a`, –Ω–æ –æ—Ç—Å—É—Ç—Å—Ç–≤—É—é—â–∏—Ö –≤ `set_b`.
–ê–Ω–∞–ª–æ–≥–∏—á–Ω–∞—è –æ–ø–µ—Ä–∞—Ü–∏—è –≤–æ–∑–º–æ–∂–Ω–∞ —Å –ø–æ–º–æ—â—å—é –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ `-`.
